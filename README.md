### CS 171

The 2015 CS 171 website is built with [Jekyll](http://jekyllrb.com).

#### Ubuntu Prerequisites

```ShellSession
$ sudo apt-get install ruby-dev
```

#### Setup

Requires Jekyll 2.1.0 or later.

```ShellSession
$ gem install jekyll
```

#### Generate and/or Serve Site

```ShellSession
$ jekyll serve --watch
```

#### View Site

```ShellSession
$ open http://0.0.0.0:4000/
```
